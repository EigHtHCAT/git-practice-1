# Задача 06 — Реализация сортировки (без встроенных)

## Участники группы
Богданов М.М.
Вахлиш Д.С.

## Описание решения
Алгоритм работает по принципу «разделяй и властвуй».
Он выбирает опорный элемент (pivot) и перераспределяет массив так, чтобы элементы меньше опорного оказались слева, а если больше, то справа.

1. Лучший случай: O(n log n): происходит, когда опорный элемент всегда делит массив ровно пополам.
2. Средний случай: O(n log n): в среднем алгоритм показывает отличную производительность на случайных данных.
3. Худший случай: O(n²): происходит, если массив уже отсортирован, а в качестве опорного всегда выбирается крайний элемент. В этом случае дерево рекурсии вырождается в список.

## Как запустить
Запустить файл в любом компиляторе C#.
