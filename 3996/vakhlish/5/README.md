# Задача 05 — Римские числа (двунаправленный конвертер)

## Участники группы
Богданов М.М.
Вахлиш Д.С.

## Описание решения
Используется тибличный метод, мы разбиваем число на разряды для работы из арабских в римские
А из римских в арабские иначе. Мы переходим по строке слева направо.
Если значение текущего символа меньше значения следующего, мы вычитаем текущее значение, а если нет, то прибавляем.
Так же есть регулярные выражения для работы.

## Как запустить
Запустить файл в любом компиляторе C#
